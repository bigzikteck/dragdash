# Dragdash 🚀
**Developed by Bigzikteck**

Dragdash is a comprehensive management and service portal designed specifically for Cyber Cafes and Grahak Seva Kendras (GSK). It streamlines the process of accessing essential digital services, providing a unified dashboard for both government and select non-government utilities.

---

## 🛠 Features
* **Government Services (GSK):** Direct access to a wide array of central and state government portals (e.g., Aadhaar, PAN, Voter ID, Ration Card, etc.).
* **Non-Government Services:** Access to curated private sector utilities portals.
* **User-Friendly Dashboard:** Simplified navigation designed for high-speed cyber cafe environments.
* **Secure Access:** Built-in protocols to ensure safe browsing and data handling during sensitive applications.

---

## 💻 System Requirements
* **OS:** Windows 10 or 11 (64-bit recommended)
* **RAM:** Minimum 4GB (8GB recommended for multitasking)
* **Internet:** Stable connection (Minimum 5 Mbps)

---

## ⚙️ Installation & Setup
1.  **Download:** Get the latest `Dragdash_Setup.exe` from the official Bigzikteck portal.
2.  **Install:** Run the installer and follow the on-screen instructions.
3.  **Authentication:** Launch Dragdash and log in using your Bigzikteck-provided Operator Activation Key.
4.  **Configuration:** * Go to **Settings > Printer Setup** to configure your thermal or A4 printers.
    * Sync your biometric device (if applicable) via the **Drivers** tab.

---

## ⚠️ Important Disclaimer
Dragdash acts as a **bridge/aggregator** between the user and service providers.
* **Government Services:** While the software facilitates access, Bigzikteck is not responsible for the approval, rejection, or processing delays of any government application.
* **Non-Gov Services:** Private services are limited and provided as-is based on third-party API availability.
